## 🛠️ Özellikler

* **Kullanıcı İşlemleri:** Giriş, Kayıt ve Şifre Yenileme sayfaları.
* **Mağaza Yapısı:** Ana sayfa, ürün listeleme ve galeri düzeni.
* **Panel & Ödeme:** Kullanıcı dashboard ekranı ve sipariş tamamlama (checkout) sayfası.
* **Tam Uyumluluk:** Tüm cihazlarda (Mobil/Masaüstü) çalışan duyarlı arayüz.

## 🚀 Teknolojiler

* **Temel:** HTML5, CSS3
* **Framework:** Bootstrap 5
* **Şablon:** SB Admin (Customized)

## 📁 Dosya Düzeni

* `index.html` / `products.html`: Mağaza ve Ürünler.
* `checkout.html`: Ödeme sayfası.
* `dashboard.html`: Kullanıcı özet ekranı.
* `login.html` / `register.html`: Kullanıcı giriş/kayıt.
* `/assets`: Stil ve görsel dosyaları.

## ⚙️ Kurulum

1. Depoyu klonlayın: `git clone https://github.com/kullanici-adiniz/proje-ismi.git`
2. Klasöre gidin: `cd proje-ismi`
3. `index.html` dosyasını tarayıcınızda açın.
